# Core Mental Models

## Knowledge → Context
Knowledge is transformed into context through retrieval and selection.

## Capability vs Control
Capability:
- Model
- Context
- Memory
- Tools

Control:
- Authentication
- Authorization
- Governance
- Evaluation
- Observability
- Human Approval

## Agent
An agent is not a smarter model.

An agent is a model operating inside a control loop.

## Learning Philosophy
- Theory before practice
- Architecture before implementation
- Mental models before tools
- Patterns before products
