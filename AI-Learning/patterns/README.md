# AI Patterns

This folder contains reusable architectural patterns rather than technology-specific notes.

Each pattern should answer:

* What problem does it solve?
* When should it be used?
* What are its strengths?
* What are its limitations?
* What components are involved?
* Where have I seen it used?

## Patterns

* Context Assembly
* Retrieval-Augmented Generation (RAG)
* Planner–Executor
* Tool Calling
* Reflection
* Human-in-the-Loop
* Memory
* Multi-Agent
* Approval Gate
* Evaluation
* Guardrails
