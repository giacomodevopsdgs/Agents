## Key Insights

### Context
Context is the task-specific information available to the model at inference time.

Context may include:
- Instructions
- User request
- Memory
- Retrieved knowledge
- Tool outputs
- Constraints

Knowledge is not context.

### Enterprise AI

Enterprise AI is not primarily a model problem.

It is a controlled capability problem.

Capability Layer:
- Model
- Context
- Memory
- Tools

Control Layer:
- Authentication
- Authorization
- Approval Gates
- Auditability
- Evaluation
- Observability
- Governance
